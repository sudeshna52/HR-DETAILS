# HR Analytics Dashboard using Power BI

## Overview

This project presents an interactive HR Analytics Dashboard built using Power BI to analyze workforce performance, salaries, departments, gender distribution, and employee demographics.

---

## Features

- Total Employees KPI
- Average Salary KPI
- Highest Salary KPI
- Average Performance KPI
- Department-wise Employee Distribution
- Salary Analysis
- Gender Distribution
- Recruitment Source Analysis
- Interactive Filters (Slicers)

---

## Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel / CSV

---

## Dataset

HR Employee Dataset
